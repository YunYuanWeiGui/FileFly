# FileFly - 基于Flask的轻量级局域网文件分享工具

> 🌐 Language / 语言选择: [English](README.md) | **简体中文**

FileFly 是一个简单、快速的局域网文件共享解决方案，基于 Python 的 Flask 框架构建。它允许你在同一网络下的设备间轻松传输文件，无需复杂的配置或外部服务。

## ✨ 功能特性

· **极简操作**：启动即用，通过浏览器即可访问和管理文件。  
· **局域网高速传输**：文件直接在本地网络传输，速度快，不消耗外网流量。  
· **跨平台支持**：基于 Web 界面，任何有浏览器的设备（电脑、手机、平板）均可访问。  
· **轻量依赖**：核心仅依赖 Flask，部署简单。  
· **目录浏览**：可浏览指定共享目录下的所有文件。

## 🚀 快速开始

1. **克隆仓库**  
   ```bash
   git clone https://github.com/YunYuanWeiGui/FileFly.git
   cd FileFly
   ```
2. **安装依赖**  
   建议使用虚拟环境。
   ```bash
   pip install flask
   ```
   注：通常仅需 Flask。请根据 requirements.txt 文件确认具体版本。
3. **运行应用**  
   ```bash
   python app.py
   ```
   运行后，控制台会显示访问地址（如 http://192.168.1.x:5000）。
4. **访问使用**  
   在同一局域网下的任何设备浏览器中，输入上一步显示的地址，即可打开文件分享界面，进行上传、下载或浏览文件。

## 📂 基本使用

· 启动服务后，默认会将运行目录下的文件共享。  
· 通过 Web 界面可以查看文件列表、下载文件。  
· （如果功能支持）部分设置可能允许通过界面直接上传文件到服务器目录。
